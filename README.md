# FIFA 23 Player Data Analysis (SQL)

## Overview
This project analyzes FIFA 23 player data using SQL to extract insights on player performance, ratings and team statistics.

## Tools Used
- SQL (SQL Server)
- Joins
- Aggregations
- Window Functions

## Key Analysis
- Player performance based on ratings
- Top players by overall score
- Team-wise performance analysis
- Ranking players using window functions

## Project File
- fifa_sql_analysis.sql

## Outcome
Extracted meaningful insights from raw data using SQL queries to support data-driven decision-making.
